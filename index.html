Bullseye Identification System for Tool Library Rack

System Overview

Three-Ring Bullseye (for tools):
  • Outer Ring: Tool Category
  • Middle Ring: Compressor Size Required
  • Inner Ring/Shape: Drive Size or Gauge (with shape identifier)
Single Circle/Shape (for accessories):
  • Nails, staples, sockets, bits only show their gauge/drive identifier

Color Key

Outer Ring - Tool Categories

Red: Nailers & Staplers
Orange: Impact Tools
Purple: Grinders & Sanders
Blue: Drills & Ratchets
Teal: Spray Equipment
Gray: Specialty Tools

Middle Ring - Compressor Size Requirements

Green: Small (1-3 CFM) - Pancake/Hotdog
Yellow: Medium (3-6 CFM) - 20-30 gal
Amber: Large (6-12 CFM) - 60 gal
Dark Red: Industrial (12+ CFM) - 80+ gal

Inner Ring - Drive/Gauge with Shapes

Nailer/Stapler Gauges (Circles)

Deep Purple Circle: 15-gauge
Navy Circle: 16-gauge
Dark Teal Circle: 18-gauge
Forest Green Circle: 11/12-gauge (framing)

Drive Sizes (Squares)

Burnt Orange Square: 1/4" drive
Crimson Square: 3/8" drive
Steel Gray Square: 1/2" drive

Implementation Notes

  • Labels: Print as stickers (2-3" diameter) for each tool and accessory bin
  • Shape System: Circles = Gauges (for nailers), Squares = Drive sizes (for impact/ratchet tools)
  • Matching: Patrons match the inner circle/square on accessories to the inner ring on tools
  • Compressor Guide: Middle ring tells them what size compressor they need to bring/rent
  • Quick Sort: Outer ring helps staff organize returns by category
  • Colorblind Friendly: Shapes provide additional differentiation beyond color
